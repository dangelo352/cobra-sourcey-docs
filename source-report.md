## Useful Maintainer-Facing Gaps

Cobra's existing README and site are strong for onboarding, but API discovery still benefits from a compact generated reference that puts command execution, validation, completion, and documentation generation in one searchable static site. The main gap this proof highlights is navigability: downstream CLI authors often need to connect concepts across several files and packages.

## Known Limits

This is an independent public documentation proof, not an official Cobra publication. The pinned commit and generated snapshot make the site reproducible, while the public source links let reviewers compare the docs against the source.
